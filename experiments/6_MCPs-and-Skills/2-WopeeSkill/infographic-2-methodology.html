Wopee.io × Claude

Evaluation Methodology & Comparison

How we measured the impact of a custom AI skill on QA automation workflows
📋
Step 1
Define Scenarios
6 real-world QA tasks: listing suites, generating coverage, running tests by ID, fetching results, re-running analysis, triaging bugs
🔀
Step 2
Run 12 Agents
Each scenario runs twice in parallel: one agent with the skill loaded, one agent without. Same model, same MCP tools, same prompt
✅
Step 3
Grade Assertions
27 assertions checked against actual tool call transcripts. Not self-reported output — real tool invocation evidence
📊
Step 4
Aggregate & Compare
Pass rates, execution time, and token usage aggregated per scenario. Statistical comparison between configurations
